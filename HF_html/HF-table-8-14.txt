  1px solid black: h1, h2 1px의 solid(점선) 검은색 밑줄 추가 한다.

	스타일들이 적용되는 방법에 대한 세상에서 가장 작고, 빠른 지침서

	1. 먼저, 임의의 선택자가 엘리먼트를 선택하나?
	  엘리먼트를 선택하는 선택자가 CSS파일에 있다면 최우선적으로 선택한다

	2. 상속은 어떻게 되나
	  일리먼트를 선택하는 선택자가 없다면 상속에 의지한다.

	3. 다 실패하면 디폴트를 사용한다
	  없다면 브라우저에 정의된 디폴트를 사용한다

	4. 여러 개의 선택자들이 하나의 엘리먼트를 선택하면 어떻게 될까
	  3가지 모두가 속해 있는 문단
	  <p class="greentea raspberry blueberry">
	  동일한 color속성을 정의 하며 이 엘리먼트와 일치하는 여러 개의 선택자들이 있는 경우, 이렇것을 충돌이라한다.
	  p { color: black;} 문단 엘리먼트
	  .greentea {color: green;} greentea 클래스 맴버 선택 선택 우선 순위가 있다.
	  p.greentea {color: green;} 좀더 특별하다
	  p.reapberry {color: blue;} 위와 동일
	  p.blueberry {color: purple;} 위와 동일

	5. class 명이 3개라면 어떻게 되나?
	  CSS 에서 정의된 가장 마지막 엘리먼트가 적용 된다 (예: p.blueberry 적용)

	/***************************** CSS 유효성 검사 *******************************/
	         	    http://jigsaw.w3.org/css-validator/

	속성 수프
       -|-color : 텍스트 엘리먼트의 폰트 색을 설정하기 위해 color을 사용
	|-font-weight : 텍스트의 굵기를 제어
	|-top : 엘리먼트의 상단 위치 설정
	|-left : 엘리먼트의 왼쪽 위치를 설정
	|-line-heigh : 텍스트 엘리먼트에 있는 줄들 사이의 공간
	|-letter-spacing : 문자 사이의 공백을 넣는다
	|-background-color : 엘리먼트의 배경색 설정
	|-border : 엘리먼트 주위의 경계선을 넣는다. (직선, 점선)
	|-margin : 엘리먼트와 콘텐츠의 경계에 공간설정
	|-font-size : 텍스트의 크기를 조정
	|-font-style : 텍스트의 글씨체 설정
	|-text-align : 왼쪽 중심 혹은 오른쪽 중심으로 텍스트 설정
	|-list-style : 리스트 아이템들이 리스트에서 어떻게 보여지는지 변경
	|-background-image : 배경 이미지를 넣을때 사용

9. 어휘력 향상시키기 : 폰트와 색으로 장식하기

  font-family속성 페이지에 있는 폰트 커스터마이징
    body {
       font-family: Verdana, Geneva, Arial, sans-serif; ((예시 자료가 필요할듯))***************************************
    }
    산세리프(Sans-serif) 패밀리 : 세리프가 없는 폰트들 테두리가 뭉글뭉글하다
    세리프(Serif) 패밀리 : 세리프 폰트를 포함하고 테두리에 가지같이 살짝 치켜 올린다
    모노스페이스(Monospace) 패밀리 : 문자들이 일정한 폭을 가지고 있다 (소프트웨어의 코드등에 쓰임)
    흘림체(Cursive) 패밀리 : 손으로 쓴듯한 필체 (제목등에 쓰임)
    판타지(Fantasy) 패밀리 : 중요한 웹디에 가끔 사용 되며 화려하다

   CSS를 사용하여 폰트 패밀리 명시하기
    body {
       font-family: Verdana, Geneva, Arial, sans-serif;
    }
    대체 되는 폰트의 리스트는 같은 패밀리에 속하여야 한다.
    , 를 기준으로 하며 대소문자를 정확히 입력 해야 한다
    좌측에서 우측으로 사용가능한 문자를 선택 하며 마지막은 디폴트 로 설정한다
    뛰어 쓰기가 있는 폰트는 "font name" 으로 둘러싼다
    
  font-size 속성을 사용한 폰트 크기의 제어
    body {
       font-size: 14px;
    }
    
  color 속성을 사용한 텍스트에 색 추가

    }

  font-weight 속성을 사용한 폰트의 굵기를 제어

  text-decoration 속성을 사용한 텍스트에 더 많은 스타일을 추가

  폰트 크기 조절하기
     px :
     body {
	     font-size: 14px;
     }
     14px: px 와 숫자 사이에 뛰우지 않는다
     px: px크기는 글자의 높이 의미한다

     % :

     %는 px를 기준으로 상대적인 크기를 가진다.

     %와는 다른 상대적인 측정 단위이ㅕ 비율을 명시 할수 없지만 척도는 가능하다.
     1.2em 은 14px의 1.2배로 약 17px의 크기를 가지게 된다.

폰트 사이즈를 어떻게 정의 해야 할가
**인터넷 익스플로어는 픽셀을 사용하여 폰트 크기를 명시할때 텍스트 척도(scaling)를 지원하지
않기 때문에 폰트의 크기를 재조정 할수 없다. 픽셀을 사용함에 따라 사용자의 접근성을 감소시키게
된다.

body {
	font-size: small; /* 디폴트로 small의 크기를 가진다 */
}
h1{
	font-szie: 150%; /* small의 150%의 크기를 h1에 적용시킨다. */
}
h2{
	font-szie: 120%; /* small의 120%의 크기를 h2에 적용시킨다. */
}
전체 적인 수정을 할때 body의 size만 조정 하면 나머지는 상속된다.

**** keyword 로 크기 제한이 싫다 ****
% or em 으로 변경이 가능 하며 통상적인 디폴트 크기16픽셀을 기준으로 한다.

    콘텐츠 영역 :

    테두리(border)는 무엇일까요?

    마진(margin; 여백)은 무엇일가여? 
      박스와 박스 사이 테두리 바깥을 둘러 싸게 된다.
