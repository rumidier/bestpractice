Features

Making slides is not a painful work!

  • Simple usage
  • Simple syntax
  • Auto generated cover page
  • Support YAML
  • Support MultiMarkdown
  • Syntax highlight for code
  • Keyboard binding (left & right arrow)
  • Written in Perl

Syntax

YAML Template

You can start from here

---
info:
  subject: All About *Silide*
  author: Keedi Kim
slides:
  - Slide 1
  - Slide 2
  - Slide 3
  - ...

info: use for slide cover

  • subject: Presentation name
  • author: Author name

How To Use?

Usage

Make symbolic link in your $PATH directory

$ cd ~/bin
$ ln -sf ~/workspace/git/tools/silide/silide.pl

Usage (contd.)

Just run silide.pl anywhere

$ mkdir -p ~/path/where/you/want/to/make/slide
$ cd ~/path/where/you/want/to/make/slide
$ silide.pl
$ ls
.
├── css
├── img
├── index.html
├── js
└── silide.yml

3 directories, 45 files
$

Edit and Run!

Edit silide.yml file, and run silide.pl again and again

$ vi silide.yml
$ silide.pl
$ vi silide.yml
$ silide.pl
$ vi silide.yml
$ silide.pl
$ ...

silide.pl will update index.html result file

More Complex Syntax

Code

Code Section

Paste your terminal code here

---
slides:
  -
    name: Code Section
    code:
      desc: Paste your terminal code here
      type: yaml
      data: |
        ---
        slides:
          - ...

code: code section

  • desc: Description for your code snippet
  • type: Kind of code to support syntax highlight
  • data: Code snippet

Code Section Example

Perl code example

---
slides:
  -
    name: Code Section
    code:
      desc: Paste your terminal code here
      type: perl
      data: |
        #!/usr/bin/env perl

        use strict;
        use warnings;

        while (<>) {
          # ...
        }

Image

Image Section

Embed image files in your slide

---
slides:
  -
    name: Image Section
    image:
      desc: Wow...
      alt: good!
      data: img/good.jpg
      attr: width=500px

image: image section

  • desc: Description for your image
  • alt: Image caption
  • data: Image path
  • attr: Attributes for image

Template Color

Change Your Color Theme

Use color section

---
info:
  ...
color:
  em: '#00ff00'
  strong: '#00ff00'
  background:
    fg: '#000000'
    bg: '#0000ff'
slides:
  - ...

color

  • em: color of em element
  • strong: color of strong element
  • background.fg, background.bg: color of background

Limitation

In Linux & Firefox 4.0

  • Limited code block size for readibility
    • width: 48
    • height: 18 ~ 24
  • Not support sequential effect yet
  • YAML & MultiMarkdown parser is not smart as you think
